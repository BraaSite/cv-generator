# CV Generator - صانع السيرة الذاتية

تطبيق ويب حديث لإنشاء سيرتك الذاتية بسهولة وبواجهة مستخدم احترافية وتفاعلية. يوفر معاينة فورية (Live Preview) مع إمكانية تصدير السيرة الذاتية كملف PDF.

## المميزات
- تصميم عصري وجذاب بتأثيرات (Glassmorphism & Gradients).
- معاينة فورية: ترى التعديلات أثناء الكتابة.
- قسم مرن للخبرات والتعليم يمكنك إضافة عدد غير محدود منها.
- طباعة وتصدير سلس إلى PDF بحجم ورقة A4.
- واجهة تدعم اللغة العربية (RTL).
- متجاوب مع جميع الشاشات.
- مبني بالكامل على HTML و CSS و JavaScript بدون أي مكتبات إضافية.

## الملفات المطلوبة
يتكون المشروع من أربعة ملفات فقط حسب الطلب:
- `index.html`: هيكل التطبيق والواجهة.
- `style.css`: التصميم والتنسيقات وأسلوب التصدير للطباعة.
- `script.js`: المنطق البرمجي للمواءمة والتحديث التلقائي بين المدخلات والمعاينة.
- `README.md`: ملف معلومات المشروع هذا.

## كيفية الاستخدام
1. قم بفتح ملف `index.html` في أي متصفح ويب حديث.
2. قم بإدخال بياناتك الشخصية، وخبراتك العلمية والعملية عبر القائمة الجانبية.
3. راقب التغييرات مباشرة تظهر على تصميم السيرة الذاتية.
4. عند الانتهاء اضغط على زر "حفظ بصيغة PDF" لتصدير وطباعة سيرتك الجاهزة.
